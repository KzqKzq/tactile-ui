#Collapsible 折叠

可展开和收起的内容区域。

#基础用法

<template>
  <Collapsible style="width: 300px;">
    <CollapsibleTrigger>点击展开/收起</CollapsibleTrigger>
    <CollapsibleContent>
      <p style="padding: 16px;">
        这是可折叠的内容区域，点击上方按钮可以展开或收起。
      </p>
    </CollapsibleContent>
  </Collapsible>
</template>

<script setup>
import { Collapsible, CollapsibleTrigger, CollapsibleContent } from '../../src/components'
</script>
