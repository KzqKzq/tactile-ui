# Switch 开关

开关用于切换单个选项的启用/禁用状态。

## 基础用法

```tsx preview
import { Switch } from '@kzqkzq/tactile-ui';

export default function SwitchDemo() {
  return (
    <div style={{ display: 'grid', gap: 12 }}>
      <Switch />
      <Switch label="启用通知" />
      <Switch label="深色模式" defaultChecked />
    </div>
  );
}
```

## 尺寸

```tsx preview
import { Switch } from '@kzqkzq/tactile-ui';

export default function SwitchSizeDemo() {
  return (
    <div style={{ display: 'flex', gap: 12 }}>
      <Switch size="sm" label="小" />
      <Switch size="md" label="默认" />
      <Switch size="lg" label="大" />
    </div>
  );
}
```

## 禁用状态

```tsx preview
import { Switch } from '@kzqkzq/tactile-ui';

export default function SwitchDisabledDemo() {
  return (
    <div style={{ display: 'flex', gap: 12 }}>
      <Switch disabled label="禁用未选中" />
      <Switch disabled defaultChecked label="禁用已选中" />
    </div>
  );
}
```

## API

### Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| checked | 是否选中（受控） | `boolean` | - |
| defaultChecked | 默认是否选中 | `boolean` | `false` |
| onCheckedChange | 状态改变回调 | `(checked: boolean) => void` | - |
| size | 尺寸 | `'sm' \| 'md' \| 'lg'` | `'md'` |
| label | 标签文本 | `string` | - |
| disabled | 是否禁用 | `boolean` | `false` |
