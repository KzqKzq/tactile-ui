# Input 输入框

输入框用于接收用户的文本输入。

## 基础用法

```vue pure
<template>
  <div class="component-showcase-vertical" style="width: 300px;">
    <Input placeholder="请输入内容" />
    <Input label="用户名" placeholder="请输入用户名" />
    <Input label="邮箱" placeholder="请输入邮箱" type="email" required />
  </div>
</template>

<script setup>
import { Input } from '../../src/components'
</script>
```

## 输入状态

```vue pure
<template>
  <div class="component-showcase-vertical" style="width: 300px;">
    <Input label="成功状态" state="success" helperText="验证通过" />
    <Input label="错误状态" state="error" helperText="请输入有效的邮箱地址" />
    <Input label="禁用状态" disabled value="不可编辑" />
  </div>
</template>

<script setup>
import { Input } from '../../src/components'
</script>
```

## 尺寸

```vue pure
<template>
  <div class="component-showcase-vertical" style="width: 300px;">
    <Input size="sm" placeholder="小尺寸" />
    <Input size="md" placeholder="默认尺寸" />
    <Input size="lg" placeholder="大尺寸" />
  </div>
</template>

<script setup>
import { Input } from '../../src/components'
</script>
```

## 多行文本

使用 `Textarea` 组件输入多行文本。

```vue pure
<template>
  <div style="width: 400px;">
    <Textarea 
      label="个人简介" 
      placeholder="请介绍一下自己..."
      :rows="4"
    />
  </div>
</template>

<script setup>
import { Textarea } from '../../src/components'
</script>
```

## API

### Input Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| label | 标签文本 | `string` | - |
| placeholder | 占位文本 | `string` | - |
| size | 尺寸 | `'sm' \| 'md' \| 'lg'` | `'md'` |
| state | 状态 | `'default' \| 'error' \| 'success'` | `'default'` |
| helperText | 帮助文本 | `string` | - |
| disabled | 是否禁用 | `boolean` | `false` |
| required | 是否必填 | `boolean` | `false` |
| iconLeft | 左侧图标 | `ReactNode` | - |
| iconRight | 右侧图标 | `ReactNode` | - |
