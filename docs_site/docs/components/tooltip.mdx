# Tooltip 文字提示

文字提示用于简短的帮助信息。

## 基础用法

```tsx preview
import { SimpleTooltip, Button } from '@kzqkzq/tactile-ui';

export default function TooltipDemo() {
  return (
    <div style={{ display: 'flex', gap: 12 }}>
      <SimpleTooltip content="这是一个提示">
        <Button>悬停查看</Button>
      </SimpleTooltip>
      <SimpleTooltip content="顶部提示" side="top">
        <Button>顶部</Button>
      </SimpleTooltip>
      <SimpleTooltip content="右侧提示" side="right">
        <Button>右侧</Button>
      </SimpleTooltip>
    </div>
  );
}
```

## API

### SimpleTooltip Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| content | 提示内容 | `string` | - |
| side | 位置 | `'top' \| 'right' \| 'bottom' \| 'left'` | `'top'` |
| delayDuration | 延迟时间(ms) | `number` | `200` |
