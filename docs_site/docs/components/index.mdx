# 组件总览

Tactile UI 提供了 **30+** 精心设计的 React 组件，涵盖表单、展示、布局、覆盖层等多种使用场景。每个组件都采用拟物化设计风格，提供丰富的交互效果。

## 基础组件

| 组件                                          | 说明           |
| --------------------------------------------- | -------------- |
| [Button 按钮](/components/button)             | 触发操作的按钮 |
| [Input 输入框](/components/input)             | 文本输入组件   |
| [Checkbox 复选框](/components/checkbox)       | 多选控件       |
| [Radio 单选框](/components/radio)             | 单选控件       |
| [Switch 开关](/components/switch)             | 切换状态控件   |
| [Slider 滑块](/components/slider)             | 滑动选择数值   |
| [Select 选择器](/components/select)           | 下拉选择组件   |
| [DatePicker 日期选择](/components/datepicker) | 日期选择组件   |
