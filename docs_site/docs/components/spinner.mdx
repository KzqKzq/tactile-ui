# Spinner 加载

加载指示器用于表示正在加载中的状态。

## 圆形 Spinner

```tsx preview
import { Spinner } from '@kzqkzq/smartisan-ui';

export default function SpinnerCircle() {
  return (
    <div style={{ display: 'flex', gap: 12, alignItems: 'center' }}>
      <Spinner size="sm" />
      <Spinner />
      <Spinner size="lg" />
    </div>
  );
}
```

## 点状 Spinner

```tsx preview
import { DotsSpinner } from '@kzqkzq/smartisan-ui';

export default function SpinnerDots() {
  return (
    <div style={{ display: 'flex', gap: 12, alignItems: 'center' }}>
      <DotsSpinner size="sm" />
      <DotsSpinner />
      <DotsSpinner size="lg" />
    </div>
  );
}
```

## API

### Spinner Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| size | 尺寸 | `'sm' \| 'md' \| 'lg'` | `'md'` |
| color | 颜色 | `'default' \| 'primary' \| 'white'` | `'default'` |
