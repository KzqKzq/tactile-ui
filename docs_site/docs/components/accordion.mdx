# Accordion 手风琴

手风琴用于折叠和展开内容区域。

## 基础用法

```tsx preview
import {
  Accordion,
  AccordionItem,
  AccordionTrigger,
  AccordionContent,
} from '@kzqkzq/tactile-ui';

export default function AccordionDemo() {
  return (
    <Accordion type="single" collapsible defaultValue="item-1" style={{ width: 400 }}>
      <AccordionItem value="item-1">
        <AccordionTrigger>什么是 Smartisan UI？</AccordionTrigger>
        <AccordionContent>
          Smartisan UI 是一个高度拟物化的 UI 组件库，灵感来自锤子科技的设计风格。
        </AccordionContent>
      </AccordionItem>
      <AccordionItem value="item-2">
        <AccordionTrigger>如何开始使用？</AccordionTrigger>
        <AccordionContent>
          只需要在项目中导入组件即可使用。所有组件都支持 TypeScript。
        </AccordionContent>
      </AccordionItem>
      <AccordionItem value="item-3">
        <AccordionTrigger>支持哪些浏览器？</AccordionTrigger>
        <AccordionContent>
          支持所有现代浏览器，包括 Chrome、Firefox、Safari 和 Edge。
        </AccordionContent>
      </AccordionItem>
    </Accordion>
  );
}
```

## API

### Accordion Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| type | 模式 | `'single' \| 'multiple'` | `'single'` |
| collapsible | 是否可折叠 | `boolean` | `false` |
| defaultValue | 默认展开项 | `string \| string[]` | - |
| value | 受控展开项 | `string \| string[]` | - |
