# Avatar 头像

头像用于展示用户头像或图标。

## 基础用法

```tsx preview
import { Avatar } from '@kzqkzq/tactile-ui';

export default function AvatarBasic() {
  return (
    <div style={{ display: 'flex', gap: 12, alignItems: 'center' }}>
      <Avatar src="https://i.pravatar.cc/100?u=1" alt="用户1" />
      <Avatar src="https://i.pravatar.cc/100?u=2" alt="用户2" />
      <Avatar fallback="张" />
      <Avatar fallback="李" />
    </div>
  );
}
```

## API

### Avatar Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| src | 图片地址 | `string` | - |
| alt | 替代文本 | `string` | - |
| fallback | 后备内容 | `string` | - |
| size | 尺寸 | `'xs' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` |
| shape | 形状 | `'circle' \| 'square'` | `'circle'` |
| status | 状态 | `'online' \| 'offline' \| 'busy' \| 'away'` | - |
| bordered | 是否有边框 | `boolean` | `false` |
