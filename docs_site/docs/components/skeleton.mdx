# Skeleton 骨架屏

骨架屏用于在内容加载时显示占位。

## 基础用法

```tsx preview
import { Skeleton } from '@kzqkzq/smartisan-ui';

export default function SkeletonBasic() {
  return (
    <div style={{ width: 320 }}>
      <Skeleton style={{ height: 20, width: '100%' }} />
      <Skeleton style={{ height: 20, width: '80%', marginTop: 8 }} />
      <Skeleton style={{ height: 20, width: '60%', marginTop: 8 }} />
    </div>
  );
}
```

## 用户骨架屏

```tsx preview
import { SkeletonUser } from '@kzqkzq/smartisan-ui';

export default function SkeletonUserDemo() {
  return <SkeletonUser />;
}
```

## 卡片骨架屏

```tsx preview
import { SkeletonCard } from '@kzqkzq/smartisan-ui';

export default function SkeletonCardDemo() {
  return (
    <div style={{ width: 320 }}>
      <SkeletonCard />
    </div>
  );
}
```

## API

### Skeleton Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| variant | 变体 | `'rect' \| 'circle' \| 'text'` | `'rect'` |
| animation | 动画 | `'pulse' \| 'wave' \| 'none'` | `'pulse'` |
