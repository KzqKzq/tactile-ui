# Separator 分割线

分割线用于分隔内容区域。

## 基础用法

```tsx preview
import { Separator } from '@kzqkzq/smartisan-ui';

export default function SeparatorBasic() {
  return (
    <div style={{ display: 'grid', gap: 8 }}>
      <p>上方内容</p>
      <Separator />
      <p>下方内容</p>
    </div>
  );
}
```

## 带标签

```tsx preview
import { Separator } from '@kzqkzq/smartisan-ui';

export default function SeparatorWithLabel() {
  return (
    <div style={{ display: 'grid', gap: 8 }}>
      <p>上方内容</p>
      <Separator label="或者" />
      <p>下方内容</p>
    </div>
  );
}
```

## API

### Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| orientation | 方向 | `'horizontal' \| 'vertical'` | `'horizontal'` |
| label | 标签文本 | `string` | - |
| decorative | 是否装饰性 | `boolean` | `true` |
