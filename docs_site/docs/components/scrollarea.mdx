# ScrollArea 滚动区域

自定义滚动条样式的滚动区域。

## 基础用法

```tsx preview
import { ScrollArea } from '@kzqkzq/smartisan-ui';

export default function ScrollAreaDemo() {
  return (
    <ScrollArea style={{ height: 200, width: 320 }}>
      <div style={{ padding: 16 }}>
        {Array.from({ length: 20 }).map((_, i) => (
          <p key={i} style={{ marginBottom: 8 }}>
            这是第 {i + 1} 行内容
          </p>
        ))}
      </div>
    </ScrollArea>
  );
}
```

## API

### Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| orientation | 滚动方向 | `'vertical' \| 'horizontal' \| 'both'` | `'vertical'` |
