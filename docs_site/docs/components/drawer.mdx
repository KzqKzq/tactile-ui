# Drawer 抽屉

抽屉从屏幕边缘滑出，用于承载更多内容。

## 基础用法

```tsx preview
import { Drawer, DrawerTrigger, DrawerContent, DrawerBody, Button } from '@kzqkzq/smartisan-ui';

export default function DrawerDemo() {
  return (
    <Drawer>
      <DrawerTrigger asChild>
        <Button>打开抽屉</Button>
      </DrawerTrigger>
      <DrawerContent position="right" title="抽屉标题" description="这是抽屉的内容区域">
        <DrawerBody>
          <p>可以在这里放置表单或说明文本。</p>
        </DrawerBody>
      </DrawerContent>
    </Drawer>
  );
}
```

## API

### DrawerContent Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| position | 位置 | `'left' \| 'right' \| 'top' \| 'bottom'` | `'right'` |
| size | 尺寸 | `'sm' \| 'md' \| 'lg' \| 'full'` | `'md'` |
| title | 标题 | `string` | - |
| description | 描述 | `string` | - |
| showCloseButton | 显示关闭按钮 | `boolean` | `true` |
