## 安装

```bash
# npm
npm install @kzqkzq/tactile-ui

# pnpm
pnpm add @kzqkzq/tactile-ui

# yarn
yarn add @kzqkzq/tactile-ui
```

## 快速上手

```tsx
import { Button, Card, CardBody } from "@kzqkzq/tactile-ui";
import "@kzqkzq/tactile-ui/style.css";

function App() {
  return (
    <Card>
      <CardBody>
        <h2>Hello, Tactile UI!</h2>
        <Button variant="primary">开始体验</Button>
      </CardBody>
    </Card>
  );
}

export default App;
```

## 设计理念

Tactile UI 是一套注重细节的拟物化设计语言。我们相信，好的界面应该有触感、有温度。通过精心调配的渐变、阴影和微动效，让数字界面也能传递真实世界的质感。

### 核心原则

- **真实感** — 模拟物理世界的光影效果
- **一致性** — 统一的设计语言和交互模式
- **可访问性** — 符合 WCAG 标准的无障碍设计
- **性能优先** — 按需加载，极致优化
